# Duckcode Observability Frontend

React TypeScript frontend for the Duckcode Observability platform.

## Tech Stack

- React 18+
- TypeScript
- TailwindCSS for styling
- React Query for data fetching
- React Router for routing
- Vite for build tooling

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
